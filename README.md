## Introduction
The User Management System with **roles** and **permissions** also supports **(Nth level) modules**.

Supports Laravel 6.x.

## Compatibility

- MySQL 8.0+
- MariaDB 10.2+
- PostgreSQL 9.4+
- SQLite 3.8.3+
- SQL Server 2008+

## Installation

    composer require paxha/lararole
